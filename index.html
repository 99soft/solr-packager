Welcome to Solr Packager!!!

Solr Packager is a small collection of Apache Maven archetypes to simplify the creation of Apache Solr standalone applications based on Apache Tomcat.

Why? Simply because we adore the KISS principle, keep it simple and straightforward, so just with a small Apache Maven magic we can easy include the Apache Solr application life cycle and deploy in your build management.

Installation

Users have not to install Solr Packager, just use it inside the Maven Archetype!

There's by the way an optional step to install the archetype, updating your local catalog, typically ~/.m2/archetype-catalog.xml:

<?xml version="1.0" encoding="UTF-8"?>
<archetype-catalog>
    <archetypes>
        <archetype>
            <groupId>com.sourcesense.solr</groupId>
            <artifactId>solr-standalone-archetype</artifactId>
            <version>1.0.0</version>
            <description>Solr Packager Standalone Archetype</description>
        </archetype>
        <archetype>
            <groupId>com.sourcesense.solr</groupId>
            <artifactId>solr-multicore-archetype</artifactId>
            <version>1.0.0</version>
            <description>Solr Packager Multicore Standalone Archetype</description>
        </archetype>
    </archetypes>
</archetype-catalog>
